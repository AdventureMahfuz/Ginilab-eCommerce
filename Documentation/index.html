HTML structure

we developed this template by using bootstrap 4. all section are in bootstrap structure. we have start and end all section by comment so that you can separate/find section easily. all section tag name we used as 'section'

logo change

you can change the logo easily. just open the index.html file in your editor and search with 'logo'. Now you are in the logo area.
if you want to use text as logo replace the 'logo here' text.

if you want to use image as logo just replace the 'heading' tag with 'img' tag (Size 155*85)

change the navigation menu

it's so easy. just open index.html file in your editor put 'main-menu' in your search bar press enter. now you can change the all menu item form here.

if you want to change mobile header (like: menu, logo, 'scroll to top' button icon) just search with 'mobile_view'. all right! your are in the right section. you can change anything from here.

[N.B you can change all the section in this process. just find the section and edit them]

CSS structure

=> CSS file (style.css) also good organized. all sections and pages are separated with comments as well index.html. just you need to see, hope you will understand easily.
file path: yourtemplate/assets/template-2/css

=> CSS file (responsive.css) it's also similar to style.css but here you will find just responsive style code for all device. and also you can change everythings.

if you want to change any kind of style you can do that. just find the specific 'class' or 'id' and change the style as you thought.

here (style.css) we store the theme-color in a variable and defined the theme color variable name as ' --primary-color: #EE3333'. if you want to change the theme color just find the line '--primary-color: #EE3333' and change the color code then you are done to change the theme color successfully.

[N.B without this two css file you don't need to work any other file.]
